# Training Schedule Telegram Bot

Бот для записи на тренировки через Telegram-группу.

## Что умеет

- тренер в личке с ботом выбирает неделю и создает расписание;
- даты, время и выходные дни выбираются через inline-кнопки;
- бот публикует красивое расписание в основную группу;
- тренер может вручную добавить участника в активное расписание;
- участники записываются одним тапом по кнопке времени;
- имена записавшихся появляются рядом со временем в опубликованном расписании;
- при первой записи бот спрашивает имя в личке и сохраняет его для будущих записей;
- запись открыта только по последнему опубликованному расписанию;
- бот отвечает через `answerCallbackQuery`, без лишних сообщений в группе;
- максимум на слот: 10 человек;
- участник может записаться на несколько тренировок;
- отмена записи идет через кнопку `❌ Отменить запись` и личное меню бота;
- история и экспорт не хранятся.

Данные пока хранятся в памяти. После перезапуска бота черновики, расписания и записи очищаются.
Имена участников сохраняются в `data/user-profiles.json` и переживают перезапуск.

## Настройка

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен.
2. Создайте файл `.env`:

```bash
touch .env
```

3. Заполните `.env`:

```bash
BOT_TOKEN=123456:token_from_botfather
GROUP_CHAT_ID=-1001234567890
TRAINER_IDS=123456789
```

Чтобы узнать свой Telegram ID, напишите боту `/whoami`.

Группа может быть приватной. Публичный `@username` не нужен: бот отправляет расписание по числовому `GROUP_CHAT_ID`.

Чтобы узнать ID приватной группы:

1. добавьте бота в группу;
2. сделайте его админом или убедитесь, что он может читать команды и писать сообщения;
3. отправьте в группе команду `/chatid`;
4. скопируйте ID в `GROUP_CHAT_ID`.

## Запуск

```bash
npm run dev
```

Для проверки TypeScript:

```bash
npm run typecheck
```

Для сборки:

```bash
npm run build
```

## Как тестить

1. Запустите бота.
2. Напишите ему `/start` в личке.
3. Нажмите `☀️ Создать расписание`.
4. Выберите неделю.
5. Выберите даты, свободное время или `ВЫХОДНОЙ`.
6. Нажмите `📣 Опубликовать расписание`.
7. В группе нажмите на слот времени, чтобы записаться.
8. При первой записи ответьте боту в личке, под каким именем вас записать.
9. Проверьте, что имя появилось рядом со временем.
10. Нажмите `❌ Отменить запись`, чтобы проверить отмену через личку.

Чтобы тренеру добавить участника вручную, напишите боту в личке:

```text
/add
```

Чтобы поменять имя для будущих записей, напишите боту в личке:

```text
/name
```
